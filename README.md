# 🖱️ Gesture Mouse Control

Controle o mouse com gestos usando visão computacional.

## 🚀 Tecnologias
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## 🎯 Funcionalidades
- Movimento do mouse com o dedo indicador
- Clique com gesto de pinça
- Scroll com gesto de dois dedos

## ▶️ Como executar

pip install -r requirements.txt
python main.py

## 💡 Sobre o projeto
Este projeto utiliza visão computacional para transformar gestos da mão em comandos de mouse em tempo real.
